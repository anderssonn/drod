Changing Settings

The Settings Screen can be reached from the Main Menu by choosing the "Settings" option.  The following settings are available:

Personal - Enter your player name here.

Graphics and Sound - Click "Use full screen" to toggle between full screen and windowed mode (640x480).  Note that you can also use the F10 key to do this at any time.  Turn sound effects and background music on or off, and set the volume each is played at.  If your sound effects or music don't work, first try closing other applications and restarting DROD.  If you are still without sound, try rebooting your computer.

Special - Select whether to play with room checkpoints or not, and whether to save demos at special points in the game. Checkpoints allow saving at certain locations in the middle of some rooms.

Editor - When "Save automatically" is checked, changes you make to a room in the editor are saved on leaving the room.  Turn this option off if you want to be prompted before saving any room changes.  Check "Show item tips" to receive help placing objects in the room.

Keyboard commands - Use to redefine the game commands to different keys.  If you are using a non-US keyboard or have selected a layout other than Qwerty (i.e. Dvorak), you may see incorrect key descriptions as you redefine keys.  Despite this, the key redefinitions will probably work once you begin playing the game.  (Sorry.  We would like to fix this problem in a future release.)

Repeat Rate - Controls keyboard repeat rate during play.  Increasing it will decrease the delay between moves when a key is held down.
