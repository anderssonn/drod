Hold Selection Screen

The Hold Selection screen is reached by choosing "Where" from the Main Menu.  This screen shows a list of all the loaded holds. Holds beaten by the current player are marked with a (*).  The following options are available:

Select hold - select a hold to play by highlighting it and pressing OK.  Pressing Cancel will leave the selected hold unchanged.

Delete hold - permanently removes a hold from this installation of DROD.  This will remove all players' saved games and demos for this hold as well, so make sure you are serious about doing this.

Export hold - saves a hold you designed to disk.  This option allows you to share your levels with other players by giving them the file saved to disk.

Import - loads a DROD data file from disk.  This allows you to load or recover exported holds and player data.
