Player Selection Screen

The Player Selection screen is reached by choosing "Who" from the Main Menu.  This screen has the following options available:

Create a new player - creates a new player charater.  This option is useful when you have more than one person playing DROD on this installation.

Select player - select a player.  Each player record contains user settings and your game progress.

Delete player - will permanently delete the selected player.  This includes deleting all of the player's saved games (so make sure this is really what you want to do!).

Export player - saves a player record to disk.  This option allows you to transfer your player settings and game information to a different DROD installation.

Import - loads a DROD data file from disk.  This allows you to recover exported player data and holds.
