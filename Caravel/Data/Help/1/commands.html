Main Menu Commands

This is a summary of the commands shown on the main menu that appear when DROD first loads:

Settings - Takes you to the Settings Screen, where you can fiddle with things.

Who - Takes you to the Player Selection Screen, where you can manage one or more player records.

Where -Takes you to the Hold Selection Screen, where you can select which levels to play.

Continue - Continues from the level and room you were at when you last exited DROD.  If you have never played the selected hold, this starts a new game from the first level.

Restore - Takes you to the Restore Screen, where you can restore a previously saved game.  Note that you can use the "Continue" command instead, but "Restore" is useful for returning to any room or level you have ever visited.

Edit - Takes you to the level editor, where you can design your own dungeons for anyone to play.

Demo - Watch a pre-recorded game demo.  Note that demos you record are only accessible from the Demos Screen.

Help - If you are reading this, then you've mastered the help command.  One caveat: if you are in fullscreen mode and go to help, it will change your screen mode back to normal (windowed).  When you are done reading help, you can press F10 to return to fullscreen mode.  For now, this is a necessary inconvenience.

Exit - You should spend all of your time playing DROD and should never have to exit the game so this command is irrelevant.
